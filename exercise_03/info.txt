Compresión de String: Implementa un algoritmo que realice una compresión básica de un string, usa un contador para los caracteres repetidos. Por ejemplo, el string aabcccccaaa se transformará en a2b1c5a3. Si el string comprimido no se vuelve más pequeño que el original, tu método deberá regresar el string original. Debes asumir que el string sólo tendrá caracteres en mayúsculas y minúsculas.
